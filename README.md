# Recetario Ecuatoriano | Ecuadorian Recipe Book

Un recetario animado bilingüe (Español / English) con 3 recetas típicas del Ecuador, presentadas en un carrusel automático que se desplaza de derecha a izquierda.

**Autor:** David Querido

## Recetas
1. Encebollado
2. Locro de Papa
3. Ceviche de Camarón

## Cómo ejecutar localmente
Abrir `index.html` en cualquier navegador moderno.

## Despliegue en GitHub Pages
1. Crear un repositorio en GitHub (ej. `recetario-ecuador`).
2. Subir todos los archivos: `index.html`, `styles.css`, `script.js`, `README.md`.
3. En el repositorio: **Settings → Pages → Source: Deploy from branch → main / root**.
4. La página estará disponible en `https://<tu-usuario>.github.io/recetario-ecuador/`.

## Tecnologías
- HTML5
- CSS3 (animaciones puras)
- JavaScript Vanilla
